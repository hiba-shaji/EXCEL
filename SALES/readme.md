Sales Distribution Dashboard
An interactive Sales Dashboard built in Microsoft Excel to analyze sales performance across regions, products, customers, and time periods.
Tools Used
- Microsoft Excel
- Power Query
- PivotTables & PivotCharts
- Slicers

Features
- KPI Cards
- Monthly Sales Analysis
- Top Products & Customers
- Regional & Country-wise Sales
- Interactive Dashboard

Dataset
Sales dataset cleaned and transformed using Power Query.
